---
title: "Break (Parar)"
---

O comando `parar` é usado para **interromper imediatamente** a execução de um loop, independentemente de sua condição.  
É útil quando você precisa **encerrar a repetição com base em uma situação específica**, como uma validação ou evento inesperado.

## Exemplo: Encerrar um loop com condição

```lipo
enquanto (verdadeiro) {
    se (condicao) {
        parar;  // sai do loop
    }
}
