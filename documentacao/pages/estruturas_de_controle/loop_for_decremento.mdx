---
title: "Loop For (para) com Decremento"
---

O loop `para` com decremento permite **percorrer valores em ordem decrescente**, ajustando o passo negativo com o operador `decremente`.  
É ideal para **contagens regressivas**, **remoção de itens**, ou qualquer situação onde o valor precisa diminuir a cada ciclo.

# Exemplo: Contagem de 10 a 0

```lipo
para i = 10 ate 0 decremente 2 faca {
    imprima i;
}
