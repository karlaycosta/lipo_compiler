---
title: "Loop While (enquanto)"
---

O loop `enquanto` é uma estrutura de repetição que **executa um bloco de código continuamente** enquanto uma **condição booleana** for verdadeira.

É ideal para situações em que **não sabemos previamente** quantas vezes o código precisa ser repetido.

## Exemplo: Contador de 1 a 5

```lipo
var contador = 1;
enquanto (contador <= 5) {
    imprima contador;
    contador = contador + 1;
}
