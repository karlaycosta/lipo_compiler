---
title: "Continue (Continuar)"
---

O comando `continuar` é usado dentro de loops para **pular a iteração atual** e **avançar para a próxima**, sem executar o restante do bloco naquele ciclo.  
É útil quando você quer **ignorar certos casos** sem interromper o loop por completo.

## Exemplo: Imprimir apenas números ímpares

```lipo
para (inteiro i = 1; i <= 10; incremente i) {
    se (i % 2 == 0) {
        continuar;  // pula os números pares
    }
    imprima i;  // imprime apenas ímpares
}
