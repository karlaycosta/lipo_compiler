---
title: "Condicional: se / senao (If/else)"
---

A estrutura condicional `se / senao` permite que o programa **tome decisões** com base em uma **expressão booleana** (verdadeiro ou falso).  
Ela é essencial para controlar o fluxo do código e executar diferentes ações conforme a situação.

## Exemplo: Verificar idade

```lipo
se (idade >= 18) {
    imprima "Maior de idade";
} senao {
    imprima "Menor de idade";
}
