---
title: "Loop For (para) com Incremento"
---

O loop `para` com incremento permite **controlar o passo da repetição**, ou seja, **quantos valores a variável de controle avança a cada ciclo**.  
Isso é útil para **pular valores**, **iterar sobre padrões específicos** ou **otimizar o desempenho**.

## Exemplo: Imprimir números pares de 0 a 20

```lipo
para i = 0 ate 20 incremente 2 faca {
    imprima i;
}
