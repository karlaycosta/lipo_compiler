---
title: "Loop For (para) Estilo C"
---

O loop `para` no estilo C oferece **controle total sobre a repetição**, permitindo definir explicitamente:

- A **inicialização** da variável de controle
- A **condição de parada**
- O **incremento ou decremento** a cada ciclo

Essa estrutura é poderosa e flexível, muito usada em linguagens como C, JavaScript e Java.

## Exemplo: Imprimir de 0 a 9

```lipo
para (inteiro i = 0; i < 10; i = i + 1) {
    imprima i;
}
